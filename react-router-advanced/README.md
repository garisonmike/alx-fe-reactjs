# React Router Advanced

This project demonstrates **advanced routing techniques** in React using React Router.  
It covers nested routes, dynamic routes, and protected routes.

## Features

- **Nested Routes**: Profile page with `details` and `settings` sub-routes.
- **Dynamic Routes**: Blog posts with variable IDs (`/blog/:postId`).
- **Protected Routes**: Some pages require authentication to access.
- **404 Page**: Catch-all route for unmatched URLs.
