# React Todo App

This project is a **Todo List application** built with React.  
It includes adding, toggling, and deleting todos, along with **unit tests** using Jest and React Testing Library.

## Features

- Display a list of todos from an initial static array.
- Add new todos with a form.
- Toggle completion status by clicking a todo.
- Delete todos individually.
- Tests for all functionalities.
