Today I am presenting my GitHub User Search React Application.

This project is a web application that allows users to search for GitHub profiles using the GitHub API.
It helps users quickly find GitHub accounts and view basic information without going directly to GitHub.

The application displays:
- Profile picture
- Username
- Bio
- Number of repositories
- Followers and following
- A button that links to the GitHub profile

This project was built using React, Tailwind CSS for styling, React Query for data fetching, and the GitHub REST API.

Now I will demonstrate how the application works.

Here is the home page of the application.
At the top there is a search input where a user can type a GitHub username.

When I type a username and click search, the app sends a request to the GitHub API and retrieves the user’s data.

Here we can see the profile image, username, bio, repository count, followers, and following.
There is also a button that opens the GitHub profile in a new tab.

If I search another username, the data updates dynamically.

If I enter an invalid username, the app shows an error message, which demonstrates error handling.

The key features of this application are:
- Real-time GitHub user search
- API data fetching with React Query
- Loading and error handling
- Dynamic UI updates
- External link to GitHub profiles

In conclusion, this project helped me learn how to work with APIs, manage server state using React Query, and build responsive user interfaces using Tailwind CSS.
